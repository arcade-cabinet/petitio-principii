# Narrative and Nonsense Generation

## Core Philosophy
The game is an exploration of circular reasoning and logical fallacies. However, relying purely on academic or Wikipedia-based definitions of debate and philosophy can lead to dry, predictable gameplay. To elevate the experience, we must embrace the absurd. 

## Challenging the "Wikipedia" Assumption
The original prompt suggested using Wikipedia and Philosophy APIs as sources for rhetoric and conjecture. We are challenging this assumption. While structural fallacies (ad hominem, petitio principii) provide the framework, the *content* of the arguments should be delightfully nonsensical.

Instead of parsing Wikipedia, we will source from **public domain surrealist works and absurdist literature** (e.g., Lewis Carroll, André Breton, Dadaist manifestos). 

## Coherent Chaining
To make the nonsense feel like a genuine argument, we need a method to "chain" it coherently.
- **Syntactic Structures:** We will map surrealist nouns, verbs, and adjectives into strict logical structures. ("If [surreal premise A], then [absurd conclusion B].")
- **Markov Chains / N-grams:** We can use lightweight Markov chains or n-gram models to ensure that the transition from one absurd thought to the next maintains grammatical and rhetorical coherence, even if the semantic meaning is entirely void.
- **Rhetorical Anchors:** Each room in the memory palace will anchor the generated nonsense to a specific rhetorical device. For instance, a "Straw Man" encounter will take a surrealist quote and deliberately misrepresent it using another surrealist quote.

By chaining surrealist text into formal logical frameworks, the game achieves its goal: a labyrinth of self-justifying nonsense.
